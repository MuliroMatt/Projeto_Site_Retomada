FAQ

Em quais plataformas o jogo está disponivel ?

O jogo está disponível apenas para computador, sendo jogado diretamente no terminal. Portanto, ele não está disponível em outras plataformas, como consoles ou dispositivos móveis.

Quais os requisitos mínimos?

Desde de que o seu computador não seja uma batata, ele provavelmente vai rodar. Talvez ele rode até mesmo em uma batata.

Como faço para baixar e instalar o jogo ?

Para baixar e executar o jogo, basta clicar no botão de download. O jogo será baixado automaticamente para o seu sistema e iniciará no terminal. Não é necessário nenhum passo adicional para jogar após o download.

O jogo oferece suporte para controle ou é jogável apenas com teclado e mouse ?

O jogo não oferece suporte para controle e é jogável apenas com teclado e mouse. Portanto, você precisará usar um teclado e um mouse para jogar, pois não há compatibilidade com controles de console ou outros dispositivos de entrada. Certifique-se de se familiarizar com os controles do teclado e mouse no jogo para aproveitar a melhor experiência de jogo possível.
